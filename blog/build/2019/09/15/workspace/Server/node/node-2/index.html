superagent / axios / node-fetch 请求数据

superagent 拓展性强，同时支持 Node.js 和浏览器，通过构建插件可以实现更多功能 ；缺点：其 API 不符合任何标准

axios 支持 Promise API，同时支持 Node.js 和浏览器

node-fetch 轻量级的模块,与window.fetchAPI 保持一致，在nodeJs中使用
